# PROJECT2025
Project Title --Weather & AQI assistant


Overview of the Project : The program is a python based command line tool that gives the real time weather and air quality updates. It eliminates the need for broweser based weather checks by fethcing data directly in terminal using Open-Meteo and Nominatim API.It not only prints raw data but give smart advices/precautions based on the data gotten from API.


Features:It gives real time weather temperature,humidity,chances of rain.It also gives real time Air Quality Index(AQI). It then after analyzing the data gives recommanded advices.It converts any city name into coordinates of any location on the Earth. It works easily without any requiring any API keys.


Technologies/tools used: This program is made using Python coding language.Also for real time data used we had used API(Application Programming Interface) which gives us real time data of weather and APIs of any location of the globe.


 Steps to install & run the project : 1. Install Python
   Make sure Python is installed on your computer.
2.Download the Code
   Download the `Project.py` file from this repository.
3. Install the Library
   Open your terminal (Command Prompt) and run this command:
   bash
   pip install requests
   Run the App Type this command in your terminal to start the bot--> python PROJECT.py
   

INSTRUCTION FOR TESTING:
Make sure you have entered the right city or town name.For more better accuracy you can input the location in city/town,(State) format. Eg. Sehore (Madhya Pradesh)
